# NarOS
#### 32位系统内核
#### 自制操作系统，主要是用来学习。

## 目前进度
### BOOT
- 原引导已废除。目前使用GRUB引导，需要支持MultiBoot
### 中断
- 中断控制器功能完全支持
- 部分中断处理不受支持
### 进程
- 支持轮询调度
- 不支持中断唤醒
- 不支持进程同步
### 内存
- 支持堆内存分配
- 支持内存保护
- 支持内存回收
### 系统调用
- 接口已实现
- 部分POSIX实现
### 保护
- 支持基本保护
- 支持内核态与用户态
### 文件系统
- 已经引入FAT16、FAT32、exFAT，但兼容层未完成
- VFS完成
### IO
- Intel处理器io指令支持
- CRT控制支持
- ATA总线支持
- PCI以及依赖PCI的设备不支持
### APIC
- 正在实现（遇到阻碍）
